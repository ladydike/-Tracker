# 샘플 데이터 폴더

이 폴더는 회의록 파일 형식과 폴더 구조의 예시를 보여줍니다.

## 📁 권장 폴더 구조

```
프로젝트루트/
├── S2_meeting/           # S2 프로젝트별 회의
├── MR_meeting/           # MR 관련 회의
├── MR_Team/              # MR 팀 내부 회의
│   ├── Team weekly/      # 팀 주간 회의
│   ├── AI Study/         # AI 스터디
│   ├── Dashboard/        # 대시보드 회의
│   └── 게임톡/            # 게임톡
└── 비정기회의/            # 비정기/기타 회의
```

## 📝 파일명 규칙

```
YYMMDD 회의명.txt
```

- 날짜는 6자리 (YYMMDD)
- 공백 후 회의명
- 확장자는 .txt

## 📄 회의록 내용 형식

자유 형식이지만, 아래 형태를 권장합니다:

```
[회의 제목]
일시: 2026-01-15
참석자: 홍길동, 김철수

[안건 1]
- 내용 1
- 내용 2

[안건 2]
- 내용 1
- 내용 2

[액션 아이템]
- TODO 1 (담당: 홍길동)
- TODO 2 (담당: 김철수)
```

## 🔄 요약 파일 생성

AI 어시스턴트에게 요약을 요청하면 `_Summary.md` 파일이 자동 생성됩니다.
