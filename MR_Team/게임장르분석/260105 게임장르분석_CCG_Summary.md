# [세션 제목: CCG(Collectible Card Game) 장르의 위기와 한계 상세 강의 노트]

## 1. 개요
* **분석 대상:** 디지털 CCG 장르 (주요 사례: 하스스톤, 유희왕, 룬테라 등)
* **세션 목적:** 하스스톤의 용어와 시스템을 통해 CCG 장르가 직면한 구조적 문제(지식의 포화, 메타 고착화)를 분석하고, 장르의 본질인 '선택의 재미'를 어떻게 복구할 것인지 논의함.

## 2. 상세 분석 내용 (주제별 정리)

### Section 1. [CCG 기초 용어 및 시스템 이해]
* **CCG vs TCG:** 거래(Trading) 가능 여부의 차이. 현대 게임은 대부분 수익 모델을 위해 거래가 불가능한 **CCG(Collectible)** 형태를 취함.
* **핵심 메카닉 (만화 기반):** 매 턴 최대 마나가 1씩 증가하며, 정해진 코스트 내에서 카드를 내는 방식. 하스스톤이 대표적임.
* **주요 용어:** 
    - **멀리건(Mulligan):** 시작 카드 교체.
    - **굿스터프(Good Stuff):** 어떤 덱에 넣어도 성능이 좋은 범용성 높은 카드.
    - **명치(Face):** 하수인을 무시하고 상대 영웅의 본체를 직접 공격하는 행위.
    - **전투의 함성(Battlecry) / 죽음의 메아리(Deathrattle):** 각각 카드를 낼 때와 죽을 때 발동하는 효과.

### Section 2. [지식의 포화와 학습 절벽]
* **핵심 문제:** 신선함을 위해 계속해서 새로운 키워드와 규칙을 추가해야 하지만, 이는 곧 유저에게 **심리적 부채**와 학습의 압박으로 다가옴.
* **야생(Wild) 시스템의 역설:** 너무 많아진 카드를 관리하기 위해 정규전에서 카드를 제외하지만, 이는 유저가 공들여 만든 카드의 가치를 훼손하고 복귀를 어렵게 만듦.
* **지식의 요구량:** CCG는 지식이 없으면 아예 시작조차 할 수 없는 구조임. '공부'가 '놀이'를 압도하는 순간 유저는 이탈함.

### Section 3. [디지털 환경에서의 정답지 고착화]
* **확률 로드맵:** 디지털 환경에서는 승률 데이터가 투명하게 공개되어 **'정답 덱(Tier 1)'**이 며칠 만에 발견됨.
* **나만의 덱(매실청 레시피) 상실:** 과거 실물 카드 시절에는 카드가 부족해 나만의 방식으로 덱을 보완하는 재미가 있었으나, 디지털에서는 모두가 정답지만을 복사하여 수행함.
* **플레이의 정형화:** 1코에 뭐 내고, 2코에 뭐 내고... 정해진 순서대로 수행만 하다가 끝나는 게임은 의외성이 거의 없으며, 이는 장르의 본질적 재미를 파괴함.

### Section 4. [CCG의 진화와 대안: 오토배틀러]
* **피로의 축 이동:** 복잡한 계산과 전투 수행은 인공지능에 맡기고, 유저는 **'상황별 선택과 전환'**에만 집중하게 함으로써 CCG의 스트레스를 완화함.
* **운과 실력의 조화:** 8인 대전 등을 통해 1:1 대결의 패배 스트레스를 분산시키고, 상황 판단력을 중시하는 구조로 재편됨.

## 3. Q&A 및 부연 설명
* **Q: 왜 하스스톤은 캐릭터만 안 바꾸고 굳이 복잡한 규칙을 계속 만드나?**
* **A:** 카드 게임은 코어 플레이의 차이가 발생하기 어려운 구조임. 아주 작은 변화로는 유저가 새로움을 느끼지 못하기 때문에, 판을 뒤엎기 위해 대량의 카드와 복잡한 규칙을 쏟아낼 수밖에 없음.
* **Q: 메타가 너무 빨리 바뀌는 건 좋은 것 아닌가?**
* **A:** 2주마다 밸런스 패치를 하면 신선함은 유지되지만, 유저가 열심히 배운 지식과 투자한 재화가 금방 쓰레기가 되는 고통을 줌. 절대 풀 수 없는 **영원한 딜레마(벼랑)**임.

## 4. 요약 및 결론
* **핵심 요약:** CCG 장르의 위기는 정해진 정답을 공부하고 수행하는 '공식화'에서 온다. **정답이 없는 환경에서의 자유로운 선택**이 주는 원초적인 재미를 복구하는 것이 장르의 숙제이다.
