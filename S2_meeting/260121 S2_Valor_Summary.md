#### 2-1. 소니 피칭 및 플레이 테스트 결과

**진행 상황:**
- 지난주 소니 대상 피칭 및 플레이 테스트 **성공적 완료**
- 선명 님 지원으로 테스트 환경 세팅 원활

**소니 측 반응:**
- 피드백 및 인게이지먼트 측면에서 **매우 긍정적**
- 게임을 재미있게 플레이함
- 개발팀(미국 매디슨 소재)이 직접 한국 방문하여 가이드 제공 → 효과적이었음

**퍼블리싱 협의 현황:**
- 소니 익스클루시브 여부: **미정** (프리 투 플레이로 최대한 많은 사람 도달 목표)
- 스테이트 플레이(State of Play) 활용 가능 여부 확인 중
- 추가 딜 협상 지속 예정

**퍼블리싱-개발팀 얼라인먼트:**
- 지난주 태석 님(커피 스튜디오)과 퍼블리싱, 개발팀 간 심층 논의
- 방향성 얼라인 진행됨

#### 2-2. 론칭 일정 및 전략

**론칭 목표:**
- **기존**: 2026년 11월 론칭 목표 (태석 님 발표 기준)
- **검토 중**: 혜성 님 요청으로 **더 빠른 론칭 가능 여부** 확인 중
- 실질적으로 필요한 것들만 챙겼을 때 11월보다 앞당길 수 있는지 체크

**론칭 방식 - 섀도 드롭(Shadow Drop) 고려:**
- 태석 님 측에서 **최대 인원에게 임팩트 있는 론칭**을 위해 제안
- **섀도 드롭**: 에이펙스 레전드처럼 사전 공개 없이 "오늘부터 플레이 가능" 방식
- 완성도 높은 게임을 갑자기 공개하는 전략

**섀도 드롭 선택 시 영향:**
- 기존 트리플A 게임 방식(1년 전 테스트 → 테크 테스트 → 하이프 빌딩) **미적용**
- 대외적 대규모 플레이 테스트 **미진행** 가능성
- MR 진행 시에도 **간소화된 내부 테스트** 수준만 가능

**테크 테스트는 필수:**
- 태석 님 강조: 대규모 인원으로 **서버/기술 테스트는 반드시 진행**
- 북미 콘솔 기준, 아웃소싱 활용 예상
- 31층 플레이 테스트 환경 스펙업 완료 → 원활히 돌아감

#### 2-3. 마일스톤 리뷰 프로세스 논의

**배경:**
- 상아 님: Valor에 MR 프로세스가 도움될 것 같아 제안
- 태석 님: 도움될 수 있다는 측면에서 논의 시작

**창한 님(CEO)의 의견:**
- 현재 프로덕션 플랜/계획 상 **HQ MR 프로세스가 부담될 수 있음**
- 플레이 테스트, 인풋 그룹 피드백 등 **파편적 의견 수렴** 정도는 가능
- 정식 MR 프로세스 태우는 것은 **부담**
- **결론**: 태석 님과 재논의 후 피드백 예정

**MR팀 권고:**
- 다음 S2 미팅에서 태석 님과 논의 결과 확인 필요
- CH님이 자주 잊으시므로 **체이스 필요**

**간소화 버전에 대한 논의:**
- 프로젝트 증가로 **정식 MR이 부담스러운 케이스** 계속 발생 예상
- "간이 버전"에 대한 니즈 증가
- 기준 및 프로세스 정리 필요
- **소유권(오너십) 문제**: 간소화 버전 어레인지 조직이 현재 MR사무국밖에 없음

**향후 방향:**
- 정식 MR 대신 **플레이 테스트 + 피드백 수렴** 방식 가능성
- 한국 퍼블리싱팀 등이 테스트할 수 있는 환경 세팅 중
- 인풋 그룹 일정은 MR과 연동되어 있으므로 **미리 일정 확정 필요**
- 3월 중순까지 방향 확정 목표
