## 2. 프로젝트별 상세 현황 (Deep Dive)

### [임팩트 (Impact) - 네온자이언트]

**Current Status**
- 개발팀이 연기 문서 작성 중 (ING 상태)
- 런칭 일정: 당초 내년 4월 목표 → 연기 불가피
- 개발팀은 추가 몇 개월 더 필요하다고 요청
- HQ 서포트 또는 추가 외주 필요 사항 있음

**Key Issues**
1. **퍼포먼스 문제**: RTX 4080 이상에서만 40프레임 수준으로 플레이 가능
2. **재미 검증 미완료**: 핵심 재미에 대한 공감대 부재
3. **들쑥날쑥한 완성도**: 시스템은 작동하나 미션/보스전 등 컨텐츠가 비어있음
4. **개발팀 자신만만함**: 근거 없는 자신감에 대한 우려
5. **반복된 연기**: 이전에도 6개월 연기 후 결과 미달성

**Discussion Points**
- **중간 검증 필요성**: 12개월 연장 요청 시에도 중간에 퍼블리싱 검증 단계 명시 필요
- **개발 우선순위 명확화 요청**: 퍼포먼스 vs 재미, 어느 쪽을 먼저 해결할 것인지 명확한 약속 필요
- **최악의 시나리오 우려**: 6개월 후에도 퍼포먼스/재미 둘 다 B- 수준으로 여전히 애매한 상황 유지
- **리스크 관리**: 크래프톤에 갚아야 할 이자가 눈덩이처럼 불어나는 상황, 캐시플로우 모니터링 중

**베타 빌드 피드백**
- 알파 대비 개선됨 - "처음으로 재밌다는 느낌" (재준 님)
- 시스템은 돌아가나 미션별 완성도 차이 큼
- 네온자이언트 입장: "퍼포먼스는 매일 잡고 있다. 컨텐츠 채우기가 우선"
- 3개월마다 빌드 나올 때 소수 인원 대비 진척은 있으나, 여전히 갈 길이 멂

**IP 매각 아이디어 (아이디어 수준)**
- 자회사가 HQ에 IP를 매각하는 방안 검토 가능
- HQ 입장: 무이자 대출보다 IP 오너십 확보가 유리할 수 있음
- 자회사 관리팀: 회수 가능성 낮은 대출 지속 시 법적 이슈(배임) 우려

### [어센트 호드 II (Ascent Horde II) - 네온자이언트]

**Current Status**
- 마일스톤 리뷰 일정: 11월 11일 (일주일 딜레이하여 잡아놓음)
- 자금 승인 대기 중 (자회사 관리팀 내부 승인 프로세스 진행)
- 1월 말 승인 → 2월 자금 집행 예정
- 3월 첫째 주 스톡홀름 방문하여 오피스 확인 예정

**Key Issues**
1. **채용 딜레이**: 당초 8명 채용 계획, 현재 개발자 거의 없음 (사라 1명 = 0.5명 수준)
2. **외주 활용 검토**: 드림루프(Dreamloop)를 어센트 호드 II에도 투입할지 고민 중
3. **킥오프 인원 구성 우려**: 코어 탤런트(예: 애니메이터) 없이 킥오프하는 게 맞는지 의문

**외주 관련**
- 드림루프: 같은 언리얼 엔진, 아이소메트릭 뷰 사용 → 효율적 협업 가능
- 2월 말 드림루프-네온자이언트 워크숍 예정
- 채용 지연 시 드림루프 활용 확대 가능성

## 3. 공통 리스크 및 인사이트

### 네온자이언트 전반 우려사항

1. **의도 vs 미인지 구분 불가**
   - 개발팀이 불편함(B, C)을 알고 선택한 것인지, 모르고 있는 것인지 판단 어려움
   - 퍼포먼스 문제: 원인을 알고 있으나 시간 부족인지, 원인 분석 자체가 안 되는 건지 불명확

2. **프로듀싱 플랜 부재**
   - 어떤 순서로 정리하고, 중간 테스트 하고, 스텝을 밟을 것인지 계획 없음
   - 민현 님: "이런 게임을 제대로 이행하는 계획이 없으면 제법이 됩니다"

3. **온도차 우려 (SDS 교훈)**
   - SDS 때 개발팀이 공감하고 대응하겠다고 했으나 실행 못함
   - 문제 영역을 감추고 있어 뒤늦게 발견한 부분 많았음
   - "큰 대형사고는 갑자기 일어나는 게 아니고, 백 건의 소규모 사건과 몇 건의 중규모 사건 후에 발생"

4. **파괴적 혁신 필요**
   - 기존 방법으로는 해결 불가, 40% 이상의 변화가 필요
   - 스튜디오 편을 들어주던 담당자들이 강하게 푸시해야 변화 가능

### 자회사 관리 이슈

- 대부분의 자회사가 돈을 빌려가는 상황
- HQ 차원에서 회수 가능성 낮은 대출 지속 시 법적 문제(배임) 우려
- 7대 3 매출 레버뉴 등 여러 방안 검토 중
