## 2. 프로젝트별 상세 현황 (Deep Dive)

### Juniper (탱고 게임웍스)

#### Current Status
- **스튜디오 헤드 교체 예정**: 3월 1일 자로 탱고 게임웍스 스튜디오 헤드가 교체될 예정
- **현 대표 → COO 전환**: 현 대표님이 회사를 떠나는 것이 아니라 COO(최고운영책임자)로 포지션을 옮겨 계속 지원 예정
- **제안서 작성 중**: 초안 리뷰 단계, 빌드 관련 내용은 아직 업데이트 미확인
- **비교 테스트 컨센서스 미팅**: 지난주 진행 예정이었으나 스튜디오 헤드 교체 이슈로 인해 진행되지 못함

#### Key Issues
1. **마일스톤 리뷰 직전 리더십 변경**: 마일스톤 리뷰까지 얼마 남지 않은 상황에서 스튜디오 동요 우려
2. **CEO 공백 가능성**: 새 CEO가 마일스톤 리뷰 전에 임명될지 불확실
3. **비즈니스 목표 변경 리스크**: 신임 CEO가 콜린 님이 작성한 비즈니스 목표에 동의하지 않을 가능성

#### Discussion Points

**스튜디오 헤드 교체 배경**
- 우석 님: "인사 관련된 내용이라 구체적으로 언급하기 어렵다"
- 창한 대표님(CH)이 의사결정하신 사항으로 알고 있음
- 마일스톤 리뷰 참석 레벨의 수정이 있을 수 있어 사전 공유

**CEO 후임 관련**
- 원래는 조 님 쪽으로 생각하고 있었으나, 내부에서 가장 적합한 인물에 대한 제안을 받을 예정
- **내부 지명 가능성 높음**: 갑자기 외부에서 CEO가 나타나는 일은 없을 것으로 예상
- 콜린 님(COO)이 계속 있으므로 수치적인 부분의 급격한 변화는 없을 것

**개발 영향도**
- 우석 님: "현 대표님이 개발 쪽에 딥하게 관여하시거나 리더십을 발휘하시는 영역은 아니었기 때문에 개발 영향은 적을 것"
- 다만 **DEV 멤버들의 심리적 영향**에 대해서는 주의 관찰 상태

**마일스톤 리뷰 진행 가능성**
- 창한 대표님이 의사결정하신 부분이므로 현 상황을 감안하고 계실 것으로 예상
- 기대치는 마일스톤 리뷰 때마다 바꿀 수 있으므로 프로세스상 큰 문제는 없을 것

### 빌드 제공 관련 논의

#### Project AA 사례 공유
- **빌드 미제공 결정**: 서브컬처 게임이고 빌드가 너무 확장되어 있어, 따로 전달 시 오해가 클 것으로 판단
- **대안**: PD님과 대표님이 인풋 그룹 대상으로 직접 발표 + 즉석 QnA (약 1시간 30분)
- **배경**: 장태석 대표님과 이미 수차례 미팅하며 빌드 조각을 보여드린 적이 있어 익스펙테이션이 어느 정도 맞춰져 있음

#### 빌드가 필요한 이유
- 문서만으로 전달되지 않는 부분들이 존재
- 개발팀이 전달하고자 하는 정보를 최대한 충실히 전달하기 위함

#### Juniper 대응 방안
- **영상 준비 중**: "하이파이 러쉬는 이랬습니다. Juniper는 이런 거예요"라는 비교 영상 준비
- 우석 님도 영상을 확인했으며, 확실히 "뭘 하고 싶었구나"라는 차이가 느껴졌다고 평가
- 발표회 형식으로 진행하는 방안에 대해 챔피언 님께 보고 예정

#### 대표님 빌드 테스트 요청
- 창한 대표님이 빌드가 있는 모든 마일스톤 리뷰 대상 프로젝트에 대해 빌드 테스트를 원하심
- 이 이슈가 Project AA의 빌드 미제공 결정의 출발점이 됨

## 3. 공통 리스크 및 인사이트

### 리더십 변경의 영향
- **단기**: 스튜디오 동요, 멤버들의 심리적 불안
- **중기**: 마일스톤 리뷰 진행의 불확실성
- **완화 요인**: 
  - 창한 대표님이 의사결정하신 사항으로 상황 감안 중
  - 현 대표가 COO로 남아 연속성 유지
  - 내부 승진 가능성 높아 급격한 변화 가능성 낮음

### 빌드 제공 vs 발표 대체
- 빌드 제공이 어려운 경우 발표회 + QnA로 대체 가능
- 핵심은 **인풋 그룹에 적절한 정보 제공** + **대표님과의 익스펙테이션 조율**

### 실험적 프로젝트 관리

- 예외 처리 시 어떤 절차와 조건이 필요한지 향후 정립 필요
