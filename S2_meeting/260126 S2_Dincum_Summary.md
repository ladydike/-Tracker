# 260126 S2_딩컴 회의록 요약

## 1. 미팅 개요

| 항목 | 내용 |
|------|------|
| **일시** | 2026년 1월 26일 (월) 오후 3:59 |
| **시간** | 약 53분 |
| **참석자** | 최지혜 (S2) |
| **주요 안건** | 오민랩(5minlab) 프로젝트 현황, 퍼블리싱 조직 변경 대응, 인력 재배치 |

**Executive Summary:**
오민랩 산하 프로젝트들(카나리아, 딩컴, 스매시레전드, 템빨용사 등)의 현황을 공유하고, 최근 퍼블리싱 담당 조직이 퍼블리싱5실로 변경됨에 따른 커뮤니케이션 공백 문제를 논의함. 특히 딩컴의 시장 검증 일정과 관련하여 퍼블리싱 측의 계획 부재로 인한 마일스톤 연기 가능성이 제기됨. 내부 엔지니어 인력난으로 인해 전면 재배치가 이루어졌으며, 4월 템빨용사 런칭 이후 인력 재분배가 예정됨. 2월부터는 오민랩 + 드림모션 두 개 스튜디오를 30분씩 나눠서 S2 미팅 진행 예정.

## 2. 프로젝트별 상세 현황 (Deep Dive)

### 2.1 카나리아 (Canary)

**Current Status:**

- 11월 킥오프 때 설정한 검증 일정대로 진행 중
- 분기별 목표를 수립하여 차근차근 개발 중
- 몬스터 종류, 스킬 성장 시스템 등 계획대로 제작 진행
- 1월 빌드: 하마토 던전 필러를 메인 텍스트와 엮어 플로우 작업 중
- 3월 빌드: 전투 시스템 완성 예정 (원신형 전투로 변경)

**Key Issues:**
- **아트 모델러 충원 난항**: 복셀 스타일 모델러 채용이 매우 어려움
  - 헤드 앤 스토마트 중심의 프로젝트 특성상 해당 스타일 전문가가 희소
  - 시장에서 수요-공급 불균형 심각 (잘하는 사람은 이미 선점됨)
  - 외주로 최대한 대응 중이나, 전담 인력 1명은 필요한 상황
  - PD(승찬 님)가 아트까지 겸해서 보고 있는 상황
- **엔지니어 인력 배분**: 카나리아에도 2~3명의 엔지니어가 필요해지면서 메인 인력 분산 필요

**Discussion Points:**
- 1월 현재 개발 속도에는 문제가 없으나, 아트 충원이 안 되면 속도 저하 우려
- 외부 채용은 상우 님 승인으로 가능하나, 지원자가 없는 것이 문제
- 작년에 전투 시스템을 원신형으로 변경하면서 경제 플로우도 재작업 중

### 2.2 딩컴 (Dinkum)

**Current Status:**
- 마일스톤 일정이 설정되어 있으나, 퍼블리싱 측 계획 미수령으로 불확실성 존재
- 작년 7월 의사결정 사항: "빌드를 빠르게 만들어 시장 검증"
- 퍼블리싱5실에서 시장 검증 계획을 가져와 실행하기로 되어 있음

**Key Issues:**
- **퍼블리싱 담당 조직 변경**: 1월 초 퍼블리싱5실로 확정되었으나, 3주간 아무런 커뮤니케이션 없음
  - 인수인계 자료 검토 중이라고 하나, 구체적인 계획 공유 없음
  - S2 입장에서는 시장 검증 속도를 맞출 수 없을 것이라는 우려
  - 퍼블리싱5실 인력 현황: 실무 3명 수준 (현호 님은 컨벤션 담당으로 제외)
- **마일스톤 연기 가능성**: 퍼블리싱 측 액션 부재 시 개발 귀책이 아닌 퍼블리싱 귀책으로 연기 프로세스 진행 필요
- **7월까지 버퍼 설정**: 박문형 대표님께 사정하여 버퍼를 확보해 놓은 상태

**Discussion Points:**
- 대표님 면담 결과: RnR로 결정된 것은 따라야 하며, 신규 담당 조직이 역할 수행해야 함
- 메일을 통해 데드라인 설정 및 퍼블리싱 측 액션 요청 필요
- 연기 프로세스 발생 시 배경 설명을 위해 MR팀 포함하여 메일 발송 요청
- 황문형 팀장님(MR팀)께도 연기 프로세스 관련 문의 → 퍼블리싱이 직접 사유 작성하거나 개발사가 사유 받아서 올리는 방식
- 이례적 케이스: 개발성 검증 마일스톤인데 사업이 할 일을 하지 않아서 연기 요청을 사업이 해야 되는 상황

**Action Required:**
- 퍼블리싱5실에 데드라인 포함한 메일 발송 (MR팀 CC)
- "이 날까지 액션이 없으면 퍼블리싱 귀책으로 일정 변경" 명시

- **엔지니어 인력난**: 4월까지 병목 구간, 내부 재배치로 대응 중
- **아트 모델러 채용난**: 복셀 스타일 전문가 희소, 외주 병행 필요

- **액션 기획자 충원 필요**: 카나리아에 액션 기획자 1명 추가 필요

- 신규 챔피언들이 게임 사업을 잘 모르는 경우가 많음

- CGPO가 풀어야 할 영역으로 인식

- MR팀(구승 님)은 공정성 이슈로 챔피언 불가

- 퍼블리싱에서 사업 계획 없이 계약만 진행하려는 경우 발생
